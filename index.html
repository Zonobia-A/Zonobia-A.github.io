如何搭建github-page

最近有想法搭建自己的blog，主要用来记录自己的学习，又苦于没有服务器，因此选择在Github上搭建自己博客。看了半天教程，踩了不少坑，总结一套比较适用新手的方法。（纯小白）

准备

  • Git

git

  1. 首先需要一个Github账号，github官网注册属于自己的Github账号。GitHub作为全球最大的程序员交友网站，是一个代码托管平台和开发者社区，开发者可以在Github上创建自己的开源项目并与其他开发者协作编码，有了Github账号，我们就可以在clone的道路上越走越远。

  2. 开始搭建博客

    首先新建一个Repositories（项目），

    千万注意项目名称命名为你的名字.github.io，然后点击create

    进入settings，确保名字是* Git_id.github.io *

    然后往下拉找到GitHub Pages

    选择theme

    填上描述，commit修改即可

    这样，一个最简单的Github Pages即搭建完成。可以在settings中的Github Pages部分点击链接访问

    点击上面链接如果能看到下面界面说明最基本的博客配置成功
